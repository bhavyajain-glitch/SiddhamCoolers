# Siddham Coolers 🌬️

A premium, full-stack E-Commerce platform for industrial and personal cooling systems. Built seamlessly with React, Express, Node.js, and MongoDB.

### 🌐 Live Hosted Application
**Frontend URL:** [https://siddham-coolers.vercel.app](https://siddham-coolers.vercel.app)  
**Backend API endpoint:** `https://siddham-coolers-api.onrender.com/api/`

---

## Technical Architecture
- **Frontend Core**: React, Typescript, Vite, Tailwind CSS, Framer Motion
- **Backend Architecture**: Node.js, Express, Typescript, OOP Strict Constructs
- **Database**: MongoDB Atlas 

## Account Roles & Authentication
The platform dynamically restricts access and renders unique dashboards purely based strictly on User Roles:
- **Customers**: Browse products, utilize coupon codes, submit orders, and view order history statuses.
- **Vendors**: Retails partners with a unique commission tracking module and specialized discount codes.
- **Administrators**: God-level access to verify stock, execute global analytics, adjust orders, and view aggregated finances.
